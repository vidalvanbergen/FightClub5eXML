# Collections

## Complete Compendiums
- **Complete_Compendium_5e** — All 5e-compatible rulebooks, third-party supplements and adventures, partnered content, Unearthed Arcana, Indie, and homebrew material.
- **Complete_Compendium_5.5e** — All 5.5e-compatible rulebooks, Unearthed Arcana, legacy content updated from the 5e sourcebooks, and homebrew material.
- **Complete_Compendium_5e+5.5e** — Merged the 5e & 5.5e compendiums from both 5e and 5.5e, with new material clearly tagged as [5e].
  - This does not include the 5e converted legacy content, for that it's best to import the complete 5e and 5e compendiums separately.

## Core Content
- **Core_Rulebooks_5e** — The 5e editions of the Player's Handbook, Dungeon Master's Guide, and Monster Manual.
- **Core_Rulebooks_5.5e** — The 5.5e editions of the Player's Handbook, Dungeon Master's Guide, and Monster Manual.
- **Core_Supplements_5e** — 5e Core Rulebooks, Xanathar's Guide to Everything, and Tasha's Cauldron of Everything.

## Adventurers League
- **Special_Adventurers_League_Forgotten_Realms_5e** — 5e DALL approved source books.
- **Special_Adventurers_League_Forgotten_Realms_5.5e** — 5.5e DALL approved source books.

## System Reference Document (SRD)
The purpose of the SRD is to provide a foundation of Dungeons & Dragons content on which third-party publishers can build their products. It allows creators to reference and use the base game rules of Dungeons & Dragons in their products without the need to pay a licensing fee to Wizards of the Coast.
- **System_Reference_Document_5e**
- **System_Reference_Document_5.5e**

## Only Official Wizards of the Coast materials
These include only the core books and official supplements.
- **WotC_5e_Only**
- **WotC_5.5e_Only**

# Additions

Starting with the official Wizards of the Coast core books and supplements, files tagged with these add the following content:

- **+PartneredOfficial** — Includes officially recognized third party content.
- **+SemiOfficial** — Includes third-party content with limited or partial official endorsement.
- **+Indie** — Independently created and published unofficial content.
- **+MercerBrew** — Includes Matt Mercer's Bloodhunter class.
- **+Homebrew** — Includes homebrew content like Matt Mercer's Bloodhunter class and fan-made or community created unofficial content.
- **+UnearthedArcana** - Officially released playtest material from Wizards of the Coast, used for testing and feedback before formal publication.
- **+Legacy_5e_All** — Includes legacy 5e content converted for use with the 5e rules.
  - **Legacy_5.5e_WotC** - Includes only the legacy corebooks and official supplements.
  - **Legacy_5e_WotC_SemiOfficial** - Includes the legacy corebooks, official supplements, and SemiOfficial content.
  - **Legacy_5e_WotC_SemiOfficial_Partnered** - Includes the legacy corebooks, official supplements, SemiOfficial, and Partnered Content.
- **\[UNTAGGED\]** - The same 5.5e Content, but without the addition of the \[5.5e\] tags.
  - The untagged files are for stand-alone use, do not use together with 5e compendium since it will override each other.
